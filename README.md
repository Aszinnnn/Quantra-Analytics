## 🌟 Sobre
Ferramenta completa para análise de dados empresariais com geração de dados fictícios, processamento e visualização.

## 🛠️ Como Usar

1. **Instalação**:
```bash
pip install pandas faker matplotlib seaborn
```
2. **Execução**
```bash
python main.py
```
